2022-07-28 (목)

ㆍ 프로젝트 주제 :
인공지능을 이용한 동아리실 출입 관리 시스템

ㆍ 개발 요소 :
Jetson Nano, Firebase, Web

ㆍ 진행과정 :
- Jetson Nano = 개발 마무리 단계 (디스플레이가 표시되는 단계만 남음)
- Firebase = DB 수립 및 적용 단계로, 개발 진행 중
- Web = 개발 마무리 단계 (학생 정보 페이지에 등록 버튼 및 차트 편집만 남음)

ㆍ 향후계획 :
- Jetson Nano 구동 개선
- Web 개발 마무리
- 학생 정보 등록
- Firebase를 통한 Database 작업 마무리

ㆍ 이후 추가할 내용 :
- 수업 중간에 나가는 학생을 체크 할 수 있도록 한다.
- 출석 명단을 엑셀로 내보내고 관리자가 보기 쉬운 UI를 만들도록 한다. (웹사이트, 앱)
- 출석, 결석 통계를 낼 수 있도록 한다.
