ㆍ 안건 :

ㆍ 진행상황 확인 :
① Jetson Nano :
- openCV의 웹 카메라의 성능 저하 문제, 인식률 문제 해결.
- Jetson Nano에 환경설정 및 프로젝트 파일을 업로드하고 정상작동 하는지 확인.

② Web :
- 회원 관리, 추가, 수정, 삭제, 회원 목록 보기 완료.
- 대시보드(차트) 완료,
- 프론트 페이지 완료.

ㆍ 개선사항 :
① PPT :
- 프로젝트 소개, 보안 관리 등 글자 크기와 문맥이 잘 맞지 않아 수정해야함.
- 프로젝트의 필요성이 분명하게 나타나 있지 않음.
- 사용기술을 젯슨나노와 웹으로 나누어 표현
- 필요없는 numpy와 같은 모듈은 삭제. 핵심 기술만 기재.

② Jetson Nano :
- 하드웨어적으로 배선, 선 관리
- 파이어베이스 → 젯슨나노 로컬 파일 전송 문제 해결.

③ Web :
- 파이어베이스 → 젯슨 나노 로컬 파일 전송 문제 해결
- 통계 차트 세분화
- 관리자 로그인 기능 추가
- 세부적인 다자인.
